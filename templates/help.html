Help

How do I request an invite?

Please enter your Reddit username and select the role which best describes you. If you're a tutor or interested in becoming a tutor, please select the tutor role. Students interested in reaching a tutor should select the student role. For all other inquiries, please select other.

How many times can I use the invite?

Invites are valid for 1 use and automatically expire after being used. If you weren't able to join the server before the invite expired, please request a new one.

I've joined the server - how can I get help?

Please state your question directly in the channel. Even if there may not appear to be anyone active, we've setup Discord to automatically notify the relevant people for new activity in the channel. Please set your Discord display name to your Reddit username.
